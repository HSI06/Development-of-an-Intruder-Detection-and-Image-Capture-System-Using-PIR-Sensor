# Intruder Detection System (PIR Sensor & Camera)

라즈베리파이 5와 PIR(인체 감지) 센서를 활용하여  
침입자를 감지하고,  
그 순간을 카메라로 촬영하여 저장하는 AIoT 보안 시스템 프로젝트입니다.

---


## 프로젝트 개요

본 프로젝트는 PIR 센서가 사람의 열원을 감지했을 때,  
라즈베리파이에 연결된 카메라를 트리거하여 실시간으로 사진을 촬영합니다.  
촬영된 사진은 발생 시각이 파일명으로 기록되어 저장되므로 보안 감시 용도로 활용 가능합니다.


---


## 주요 부품 설명
### PIR 센서란?  
사람이나 동물 등 열을 방출하는 물체의 움직임을 감지하는 센서임  
적외선을 직접 발사하지 않고, 주변의 적외선 변화를 수동적으로 감지함  
화용 예시 : 자동 조명, 침입 감지/보안 시스템, 에너지 절약 시스템, 스마트홈 자동화 등  


V : VCC

G : GND

S : Signal(디지털 출력)

출력 방식 : 디지털 (HIGH/LOW)

동작 전압 : 3.3V ~ 5V

감지 범위 : 약 3 ~ 5m

감지 각도 : 약 100°

반응 시간 : 약 2초 이내


---


## 주요 기능

실시간 움직임 감지: PIR 센서를 통해 적외선 변화(열원 이동)를 상시 모니터링합니다.

자동 사진 촬영: 감지 신호(High)가 들어오는 즉시 카메라 Preview를 생성하고 사진을 캡처합니다.

스마트 파일 관리: datetime 라이브러리를 통해 촬영된 시각(YY-MM-DD HH:M:S.jpg)을 파일명으로 자동 지정하여 저장합니다.
